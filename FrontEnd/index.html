What's All Abroad?

All Abroad is a web application for students who are planning to or are already studying abroad. This is all-in-one kind of application which includes various features which seem to be important to abroad students. Weather, Translator, and Chat features are all part of All Abroad aimed to help abroad life. The Chat feature allows nearby abroad students to connect with each other.
